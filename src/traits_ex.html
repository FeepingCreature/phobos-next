  • uniqueEnumMembersHashed

    Declaration

    auto uniqueEnumMembersHashed(T)() if (is(T == enum));

    Hash-table version of uniqueEnumMembers.

    Examples

    1. enum E { x, y, z, Z = z, Y = y } import std.algorithm.comparison : equal; assert(uniqueEnumMembers!E.equal([E.x, E.y, E.z])); // run-time static assert(uniqueEnumMembers!E.equal([E.x, E.y, E.z])); // compile-time static assert(E.x == 0); static assert(E.y == 1); static assert(E.z == 2); static assert(E.Z == E.z); static assert(E.Y == E.y); static assert(uniqueEnumMembers!E.equal(uniqueEnumMembersHashed!E));
