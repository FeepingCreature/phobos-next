  • Array

    Declaration

    struct Array(E, Assignment assignment = Assignment.disabled, Ordering ordering = Ordering.unsorted, bool useGCAllocation = false, alias less = "a < b");

    Array of value types E with optional ordering given by ordering.

    Parameters

    true iff GC.malloc is used for store allocation, otherwise C's {m,ce,re}alloc() is used.
